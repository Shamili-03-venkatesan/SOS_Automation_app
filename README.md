🚨 SOS Automation Android Application 🚨

* The **SOS Automation Android Application** is a personal safety application designed to help users quickly alert a trusted guardian during emergency situations.
* When the **SOS feature** is activated, the application sends an alert containing the user's **name, guardian number, native place, real-time live location (Google Maps link), and a captured photo** 📸.
* The application ensures **fast communication through SMS and WhatsApp 💬**, while protecting user privacy with **secure permission-based access 🔒**.

---

## ✨ Features

* 🚨 **Instant SOS Alert** – Sends emergency alert messages immediately to the registered guardian.
* 📍 **Live Location Sharing** – Shares the user’s real-time location using a **Google Maps link**.
* 📸 **Automatic Photo Capture** – Captures the user's image during the SOS activation for identification.

---

## ⚙️ How It Works

1. The user registers their **name, guardian contact number, and native place** in the application.
2. In an emergency, the user **presses the SOS button** in the app.
3. The application **captures the current photo and GPS location** of the user.
4. The system **sends an alert message with details and location link** to the guardian through **SMS and WhatsApp**.

---

## 🛠️ Technologies Used

* **Java & Android Studio** – For developing the Android mobile application.
* **Google Maps API & GPS Services** – For real-time location tracking and map link generation.
